# Value of Growth Calculation

## Given Data

| Variable | Value | Description |
|----------|-------|-------------|
| Growth CapEx | €100,000 | Money invested in expansion |
| ROI | 25% | Return on investment from growth initiatives |
| WACC | 10% | Weighted average cost of capital |

## Step-by-Step Calculation

### Step 1: Calculate Net Incremental Earnings

```
Net Incremental Earnings = Growth CapEx × (ROI - WACC)
Net Incremental Earnings = €100,000 × (25% - 10%)
Net Incremental Earnings = €100,000 × 15%
Net Incremental Earnings = €15,000
```

**Interpretation:** The company earns €15,000 MORE than what investors require (the excess return).

### Step 2: Calculate Value of Growth (Method 1)

```
Value of Growth = Net Incremental Earnings / WACC
Value of Growth = €15,000 / 10%
Value of Growth = €150,000
```

### Step 3: Calculate Value of Growth (Method 2)

```
Value of Growth = [Growth CapEx × (ROI - WACC)] / WACC
Value of Growth = [€100,000 × (25% - 10%)] / 10%
Value of Growth = [€100,000 × 15%] / 10%
Value of Growth = €15,000 / 10%
Value of Growth = €150,000
```

## Why This Makes Sense

| Item | Amount | Calculation |
|------|--------|-------------|
| You invested | €100,000 | Growth CapEx |
| You earn | €25,000/year | 25% × €100,000 |
| Investors expect | €10,000/year | 10% × €100,000 |
| **Excess earnings** | **€15,000/year** | €25,000 - €10,000 |
| **Value of perpetual excess** | **€150,000** | €15,000 / 10% |

## The Three Equivalent Formulas

All three formulas give the same answer: **€150,000**

### Formula 1: Using Net Incremental Earnings
```
Value of Growth = Net Incremental Earnings / WACC
Value of Growth = €15,000 / 10% = €150,000
```

### Formula 2: Direct Calculation
```
Value of Growth = [Growth CapEx × (ROI - WACC)] / WACC
Value of Growth = [€100,000 × 15%] / 10% = €150,000
```

### Formula 3: Multiplier Approach
```
Value of Growth = Growth CapEx × [(ROI - WACC) / WACC]
Value of Growth = €100,000 × [15% / 10%]
Value of Growth = €100,000 × 1.5 = €150,000
```

## Real-World Interpretation

By investing **€100,000** in growth at **25% ROI** (when capital costs only **10%**), the company creates **€150,000** in value for shareholders.

This is **€50,000 more** than the €100,000 invested because the excess returns compound perpetually.
