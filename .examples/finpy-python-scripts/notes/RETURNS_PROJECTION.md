# Returns Projection - €100,000 Investment

**Assumptions:**
- Starting Capital: €100,000
- Annual Return: 22% (based on Top 25 portfolio historical performance)
- Tax Rate: 16% (on realized gains only)
- Annual Rebalancing: 30% of positions closed yearly
- Tax Applied: Only on the 30% of gains realized each year
