**Unique differentiation: V7 Fund Scoring**

Apply V7 Capital framework (companies) to **funds themselves**:

```
Fund Score = GP Track Record (50%) + Timing (20%) + Capital Efficiency (15%)

Output: INVEST / PASS / REJECT + Intrinsic Fund Value
```

**Nobody does this.** VCs score companies. LPs need to score funds.
